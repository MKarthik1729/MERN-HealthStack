HealthStack

Make sure to install node
Install dependencies

- To open Admin Dashboard Open a new terminal
cd /admin
npm isntall
npm start

- To open main HealthStack website Open another new terminal
cd /client
npm install
npm start

- To run server Open another new terminal
cd /server
npm install
nodemon server
